      Instruções de Uso

      - Upload: Clique para adicionar arquivos. Volte à raiz para validar.
      - Nova Pasta: Crie uma nova pasta. Volte à raiz para validar.
      - Tema: Alterne entre claro e escuro.
      - Download: Baixe um arquivo. Volte à raiz para validar.
      - Excluir: Remova um arquivo ou pasta.
      - Renomear: Altere o nome de um arquivo ou pasta.
      - Visualizar: Veja o conteúdo de um arquivo.
      - Arrastar: Arraste arquivos para qualquer pasta.
      - Navegação: Clique duas vezes em uma pasta para entrar, ou em "Raiz" para voltar.
